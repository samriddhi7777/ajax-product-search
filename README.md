# AJAX Product Search System (PHP + MySQL)

## Features
- Live search using AJAX
- No page reload
- Dynamic results
- MySQL database integration

## Technologies Used
- PHP
- MySQL
- AJAX
- HTML

## How to Run
1. Install XAMPP
2. Place folder in htdocs
3. Start Apache & MySQL
4. Open:
   http://localhost/ajax-product-search

## Note
This project runs on a local server (XAMPP)
